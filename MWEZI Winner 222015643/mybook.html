      7. Attributes

        8. Attributes of customer Table
        Here are the descriptions or purposes of the attributes/columns used in customer table:

        customer_id (primary key): id for the customer.

        First_name(varchar): first name of customer.

        Last_name(varchar): last name of customer.

        age(varchar): age of customer.

        phone number(varchar): telephone number of customer.

        creditcard(varchar): creditcardnumber of customer.

        Attributes of artist Table
        Here are the descriptions or purposes of the attributes/columns used in artist table:

        artist_Id: The column which are represents id of artist .
        AUTO_INCREMENT means that the database will automatically generate a unique ID for each new record, typically incrementing the previous value of artist table.

        names(varchar) the column which are represents names of artist


        age(varchar): The column which are represents age of an artist.

        gender(varchar): The columns which are represents gender of an artist.

        nationality(varchar): The columns which are represents country where an artist came from.

        experiance(varchar): The columns which are represents experiance of an artist or how many year he/she worked .

        Attributes of warehouse table
        Here are the descriptions or purposes of the attributes/columns used in warehouse table:

        warehouse_Id(primary key): The column which are represents id of warehouse which is unique value called PRIMARY KEY. It is set to auto-increment (AUTO_INCREMENT).
        AUTO_INCREMENT means that the database will automatically generate a unique ID for each new record, typically incrementing the previous value of warehouse.


        warehouse_location(varchar): The column which are represents name of warehouse location.


        Attributes of order Table
        Here are the descriptions or purposes of the attributes/columns used in order table:

        order_id(primary key): The column which are represents id id which is unique value called PRIMARY KEY. It is set to auto-increment (AUTO_INCREMENT).
        AUTO_INCREMENT means that the database will automatically generate a unique ID for each new record, typically incrementing the previous value of order table.


        date_and_time(varchar): The column which are represents the time customer ordered something


        Attributes of portrait Table
        Here are the descriptions or purposes of the attributes/columns used in portrait table:

        portrait_id (primary key): The column which are represents id of portrait which is unique value called PRIMARY KEY. It is set to auto-increment (AUTO_INCREMENT).
        AUTO_INCREMENT means that the database will automatically generate a unique ID for each new record, typically incrementing the previous value of portrait table.


        portrait_name(varchar): The column which are represents name of the portrairt.


        cost(varchar): The column which are represents the cost of portrait/paint.
